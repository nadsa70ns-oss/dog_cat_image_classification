# dog_cat_image_classification
Dog and Cat image classification using Google Teachable Machine and TensorFlow (Keras).

# Image Classification Project

## 1. Create Model
I used Teachable Machine to create an image classification model with two classes:
- Cat
- Dog

## 2. Train Model
I trained the model using multiple images for each class.

## 3. Export Model
I exported the model as:
- keras_model.h5
- labels.txt

## 4. Python Code
I wrote a Python script (predict.py) to:
- Load the model
- Read an image
- Predict the class

## 5. Test
I tested the model using an image (j1.png)
